## Deployment

### Prerequirement

- [Node.js](https://nodejs.org/)
- [Git](https://git-scm.com/)


1. **Cloning the repo :**
    ```sh
    git clone https://github.com/yrieix-cisterne-nws/cash-stadium.git .
    cd cash-stadium
    ```

2. **Launch in Local :**
    ```sh
    npm run dev
    ``` 
 
3. **Localhost**

Website should be visible on 
`https://localhost:3000`
