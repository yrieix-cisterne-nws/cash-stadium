## Deployment

### Prerequisites

- [Node.js](https://nodejs.org/)
- [Git](https://git-scm.com/)

### Local

1. **Clone the repository:**
```sh
    git clone https://github.com/yrieix-cisterne-nws/cash-stadium.git
    cd cash-stadium
```

2. **Launch locally:**
```sh
    npm run dev
```

3. **Localhost**

   The website should be accessible at `http://localhost:3000`
